# 隱私權政策

> **[English Privacy Policy](../PRIVACY.md)** — Read this privacy policy in English

**最後更新：** 2026-07-16

DS studio（以下稱「本擴充功能」）致力於保護您的隱私。本政策說明本擴充功能存取哪些資料、如何使用，以及您對自身資料的選擇。

## 資料收集

**本擴充功能不會收集、傳輸或出售任何個人資料。**

本擴充功能處理的所有資料均儲存在**您的本機裝置**或**Chrome 瀏覽器內建的同步基礎架構**中。沒有任何資料會傳送至開發者的伺服器或任何第三方。

## 權限與資料使用

### `storage`

用於儲存您的設定、提示詞組與擴充功能資料。

- **`chrome.storage.local`** — 儲存提示詞組內容、UI 調整偏好、審查回覆恢復記錄與操作狀態。這些資料不會離開您的裝置。此外，最近一次擷取到的 DeepSeek 登入權杖（`dss-last-auth-token`）也僅儲存於此——**此權杖永遠不會同步，只留在擷取到它的那台裝置上**，用於在瀏覽器重新啟動時補刪殘留的臨時對話。
- **`chrome.storage.sync`** — 儲存提示詞組名稱、啟用/停用開關與非敏感性設定，並包含「臨時對話待刪佇列」（`dss-pending-deletes-sync`）。此佇列僅記錄**非敏感性**的對話代號與重試次數（`chatUuid`、`attemptCount`），**不含任何權杖或對話內容**，作為跨裝置補救刪除的單一事實來源，讓您在其他登入同一 Chrome 帳戶的裝置上也能清除意外殘留的臨時對話。這些資料會透過您的 Google 帳戶在 Chrome 瀏覽器之間同步。開發者無法存取這些資料。

### `activeTab`

當您點擊擴充功能圖示開啟彈出選單時使用。此權限允許彈出選單讀取當前活躍分頁的網址，以判斷您是否位於 `chat.deepseek.com`。不會讀取或傳輸任何分頁內容。

### `alarms`

用於排程背景任務，例如重試失敗的臨時對話刪除。此過程中不會收集任何資料。

### Content Scripts

本擴充功能會將 content script 注入 `chat.deepseek.com` 以提供各項功能（提示詞注入、UI 調整、審查回覆恢復等）。這些 script：

- 讀取並修改對話頁面 DOM 以增強使用者介面。
- 攔截來自 DeepSeek 伺服器的網路回應，目的僅為還原被審查屏蔽的回覆。
- 讀取 DeepSeek 網頁自身儲存於您裝置本機的 IndexedDB 對話快取（`deepseek-chat` 資料庫的 `history-message`），僅用於「完整對話歷史面板」在本機呈現、搜尋與匯出您目前的對話。此讀取完全在本機進行，**不會**為此發出任何網路請求，資料也不會離開您的裝置。
- **不會**在您的瀏覽器之外讀取、收集或傳輸任何資料。

## 網路請求

本擴充功能**僅**向 `chat.deepseek.com` 發出網路請求，且僅在您明確使用的功能情境下：

- **臨時對話刪除** — 在您刪除臨時對話時向 DeepSeek 的 API 發送刪除請求。
- **審查回覆恢復** — 攔截 DeepSeek 的串流回應以還原被審查的內容。

這些請求是代您發出的，功能上與 DeepSeek 網頁應用原生的行為完全相同。

**本擴充功能從不與開發者控制的任何伺服器通訊。**

## 臨時對話的跨裝置補救刪除

為了讓「臨時對話」功能在分頁/瀏覽器被直接關閉、甚至電腦被強制關機的情況下仍能盡力清除殘留對話，本擴充功能採用以下資料處理方式：

- **待刪佇列（`chrome.storage.sync`，鍵值 `dss-pending-deletes-sync`）** — 僅記錄非敏感性的 `chatUuid`（對話代號）與 `attemptCount`（重試次數），供任一登入同一 Chrome 帳戶的裝置在下次啟動時讀取並補刪。
- **登入權杖（`chrome.storage.local`，鍵值 `dss-last-auth-token`）** — 僅儲存於本機，**從不透過 `chrome.storage.sync` 同步或傳送至任何裝置**。補救刪除一律使用「執行補救的那台裝置自己擷取到的權杖」，而非跨裝置傳遞權杖。
- **裝置本機的開啟中對話清單（`dss-open-temp-uuids`，僅本機儲存）** — 記錄目前這台裝置正在開啟中的臨時對話，確保跨裝置同步觸發的補救掃描**永遠不會刪除您正在使用中的對話**。

## 第三方服務

本擴充功能**不**使用任何第三方分析、當機回報、遙測或廣告服務。除了擴充功能自身打包的 script 之外，執行時期不會載入任何第三方程式碼。

## 資料保留

- **本機資料** — 持續保留直到您解除安裝擴充功能或清除該擴充功能的瀏覽器資料。
- **同步資料** — 持續保留在您的 Chrome 同步帳戶中，直到您解除安裝擴充功能、關閉 Chrome 同步或清除同步資料。

## 您的選擇

- **解除安裝** — 從 Chrome 移除擴充功能會刪除所有本機儲存的資料。同步資料可能會在 Chrome 同步帳戶中保留至下一次同步週期。
- **Chrome 同步** — 您可以在 Chrome 設定中關閉擴充功能的 Chrome 同步，以防止設定跨裝置同步。
