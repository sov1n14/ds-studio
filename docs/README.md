# DS studio

> **[English README](../README.md)** — Read this project's documentation in English

DS studio 是一個 Chrome 擴充功能，專為優化 `chat.deepseek.com` 的使用體驗而設計。您可以建立多組提示詞組，在發送訊息前自動將選取的提示詞注入到輸入內容中，並針對不同場景快速切換。

## 功能一覽

| 類別 | 功能 |
|-|-|
| **提示詞系統** | 多組提示詞管理、自動注入、全域預設提示詞、對話綁定、空白選項模式 |
| **快速切換** | 頁面內 Overlay 下拉選單，支援搜尋篩選與拖曳排序，與彈出選單雙向同步 |
| **UI 調整** | 側邊欄自動隱藏、對話/輸入框寬度調整、隱藏思考過程、回到頂部按鈕 |
| **對話匯出** | 一鍵匯出完整對話為 Markdown，支援思考過程與參考連結開關 |
| **完整對話歷史面板** | 讀取本機完整對話、繞過 DeepSeek 虛擬列表無法捲到頂的缺陷，可到達最舊訊息，支援全文搜尋與 Markdown 匯出 |
| **引用回覆** | 選取 AI 回覆文字後一鍵轉為 Markdown 區塊引用 |
| **系統時間注入** | 自動於每則訊息附加目前系統時間與時區偏移 |
| **審查回覆恢復** | 自動從串流資料還原被 DeepSeek 審查屏蔽的原始回覆，跨重整保留 |
| **雲端同步** | 跨裝置自動同步提示詞組與設定，內建衝突偵測與智慧合併 |
| **JSON 備份與還原** | 完整備份與還原提示詞組、設定，以及審查回覆恢復記錄 |
| **行動裝置** | 側欄滑動手勢，解決行動版缺乏側邊欄快速切換的問題 |

詳細功能說明請見 [FEATURES.md](FEATURES.md)。

## 安裝說明

### 透過 Chrome 擴充功能頁面載入（開發者模式）

1. 下載或複製本專案至本機。
2. 開啟 Chrome 瀏覽器，在網址列輸入 `chrome://extensions/` 並前往。
3. 開啟右上角的「**開發人員模式**」（Developer mode）開關。
4. 點擊左上角出現的「**載入未封裝項目**」（Load unpacked）按鈕。
5. 在彈出的檔案選擇視窗中，選取專案根目錄（包含 `manifest.json`）。
6. 載入成功後，擴充功能會出現在列表中，並在瀏覽器工具列顯示圖示。

### 安裝後注意事項

- 安裝完成後，建議重新整理已開啟的 `chat.deepseek.com` 分頁，以確保 Content Script 正確載入。
- 如果擴充功能圖示未出現在工具列，可點擊右側的拼圖圖示，找到 DS studio 並釘選。

## 開發技術

此專案採用 Manifest V3 開發，符合 Chrome 擴充功能的官方安全規範。所有 DOM 攔截邏輯皆運作於 Content Script 中，確保注入的內容能正確觸發目標網站的狀態更新。支援 `chrome.storage.sync` 跨裝置同步，並內建衝突偵測與智慧合併機制。
